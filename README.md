## instalation

`install_mac.sh` or `install_linux.sh`

after installation it paste to the vim

```
:Helptags
```

## how to use

### navigation
```
F3
```

### comment
```
\ci
```

### go to definition
- Horizonal split
```
\dd
```
- Vertical split
```
\sd
```
- New tab
```
\td
```

### choose window
```
- [window or tab num]
```

### swap windows
```
-s [window to swap]
```

### resize windows
```
^C[hjkl]
```

### new tab
```
tt
```

### move lines
select lines and:
```
Shift[kj]
```
